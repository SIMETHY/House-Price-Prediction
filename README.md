# 📊 Machine Learning Projects

This repository contains multiple machine learning projects focused on regression analysis, data preprocessing, and predictive modeling.


## 📁 Projects Included

A machine learning project to analyze retail data and predict sales and profit based on various features.

#### 🔹 Features
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Univariate & Bivariate Analysis  
- Outlier Detection & Handling  
- Log Transformation  
- Feature Scaling  
- Model Training (Regression Models)  
- Performance Evaluation  

#### 🔹 Techniques Used
- Linear Regression  
- Logistic Regression (if applicable)  
- Data Visualization (Matplotlib, Seaborn)
